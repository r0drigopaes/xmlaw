DungeonsOfHack applet

A Gauntlet clone with generated levels. Pick up all the gold in each level.
Keys can be used to open doors (the brown wall segments are doors).
Move around using the cursor keys. Use W,S,A,D to shoot.
Use the Shift key plus a direction to open a door.
This game's levels should still be made progressively harder.
(as the difficulty level now randomly ranges from "pretty challenging" to "very difficult").
